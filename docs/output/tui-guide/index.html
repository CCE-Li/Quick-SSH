TUI 界面指南

Quick-SSH 的 TUI 界面基于 ratatui 和 crossterm 构建，采用事件驱动架构，提供高效的键盘操作体验。

界面布局

TUI 界面由四个部分组成：

┌─ 标题栏 ──────────────────────────────────────────┐
│ Quick-SSH v2.0.1  |  共 5 台主机  |  模式: NORMAL  │
├──────────────┬──────────────────────────────────────┤
│              │                                      │
│  主机列表    │           详情面板                    │
│              │                                      │
│  ○ mysrv     │   别名: mysrv                        │
│  ● devbox    │   地址: ********                     │
│  ○ web-prod  │   密钥: ~/.ssh/id_rsa               │
│  ○ staging   │   状态: ● 在线                       │
│              │                                      │
├──────────────┴──────────────────────────────────────┤
│ 状态栏: j↓ k↑ gg↕ G↕ /搜索 a添加 e编辑 ...         │
└─────────────────────────────────────────────────────┘
  • 标题栏：显示版本号、主机数量、当前模式
  • 主机列表：显示所有 SSH 主机及其在线状态
  • 详情面板：显示选中主机的详细信息
  • 状态栏：显示当前模式的操作提示或闪烁消息

主机列表前缀含义

图标含义
○未检测
● (绿色)在线
● (红色)离线
◔检测中
> 已标记

交互模式

TUI 支持多种交互模式，通过状态栏可识别当前所处模式：

  • Name
    NORMAL
    Type
    默认模式
    Description

    浏览主机列表，可使用所有导航和操作快捷键

  • Name
    SEARCH
    Type
    搜索模式
    Description

    输入关键词过滤主机列表，实时筛选

  • Name
    ADD / EDIT
    Type
    新增/编辑模式
    Description

    弹窗表单编辑主机字段，支持 Tab 切换字段，Ctrl+S 保存

  • Name
    CONFIRM
    Type
    确认模式
    Description

    删除确认对话框，按 y/Y 确认，n/N/Esc 取消

  • Name
    HELP
    Type
    帮助模式
    Description

    显示键盘快捷键帮助弹窗，按 q/Esc 关闭

完整快捷键

导航

按键功能
j / ↓向下移动
k / ↑向上移动
gg跳到顶部
G跳到底部

主机操作

按键功能
Enter连接选中主机
Space标记/取消标记
a新增主机（弹窗表单）
e编辑当前主机（弹窗表单）
d删除选中主机
.切换地址显示/隐藏（隐私保护）

检测

按键功能
pPing 检测当前主机
PPing 检测所有主机

搜索

按键功能
/进入搜索模式
Enter确认搜索
Esc取消搜索

系统

按键功能
q退出 TUI
?显示帮助
Esc取消/返回

新增/编辑主机弹窗

按 a 或 e 会打开主机编辑弹窗，包含以下字段：

字段说明
Host主机别名（必填）
HostName主机地址（IP 或域名）
User登录用户名
PortSSH 端口（默认 22）
IdentityFile密钥文件路径（自动预填充默认路径）
其他指令 / 注释多行文本，可添加额外 SSH 配置或注释

弹窗操作

按键功能
Tab切换到下一字段
Shift+Tab切换到上一字段
Enter跳转到下一字段（仅单行字段）
Ctrl+S保存并关闭弹窗
Esc取消编辑

"其他指令 / 注释"字段为多行模式，可用于添加 ServerAliveInterval、ProxyJump 等不被 qssh 直接管理的 SSH 指令。但 HostName、User、Port、IdentityFile 等字段禁止在此处重复填写。

连接主机

在 NORMAL 模式下选中主机后按 Enter 键即可连接。连接流程：

  1. TUI 退出备用屏幕，恢复终端
  2. 启动 SSH 交互式会话（stdin 继承，密码认证正常工作）
  3. SSH 退出后自动重新进入 TUI 界面

拖拽上传

在 SSH 会话期间，将文件拖入终端窗口即可触发文件上传。详见 文件上传。
